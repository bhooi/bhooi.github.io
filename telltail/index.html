TellTail: a Principled Scoring Function for Dense Subgraphs

Overview

TellTail is a scoring function for identifying surprisingly dense subgraphs.
TellTail has the following properties:
  • Accuracy: accurately identifies blocks which are dense internally and sparsely connected externally, based on two empirical patterns we observe about dense subgraph distributions in real graphs
  • Fast computation: runs in linear time in the data size. TellTail avoids sampling and hence provides large speedups over alternative methods.
  • Theoretically sound: the score is based on estimating the probability of a subgraph of the same size having larger mass than the current block. We also show consistency guarantees.
